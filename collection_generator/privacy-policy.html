Privacy Policy - CollectGen

Last updated: March 31, 2026

CollectGen ("we", "our", or "the extension") is a Chrome extension that captures API request metadata while you browse and allows you to export that data as a JSON collection file.

1. Data We Process

When capture mode is enabled by you, CollectGen may process:

  • Request URL
  • HTTP method
  • Request headers
  • Request body (when available)
  • Response status code
  • Capture timestamp

2. How Data Is Stored

Captured data is stored locally in your browser using chrome.storage.local. Data remains on your device unless you explicitly export it.

3. Data Sharing and Transfers

CollectGen does not sell, rent, or transfer your data to third parties. We do not send captured API data to our servers in the current free version.

4. Purpose of Data Use

Data is used only for the extension's single purpose: capturing API request information and exporting it as a collection JSON file for developer/testing workflows.

5. User Control

  • You can start or stop capture at any time from the extension popup.
  • You can clear captured data using the Clear button.
  • You can export captured data manually using Export Collection JSON.

6. Security

We follow Chrome Extension platform controls and least-privilege practices for the current feature set. However, no software can guarantee absolute security.

7. Children's Privacy

CollectGen is not directed to children under 13 and is intended for developer/testing use.

8. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top indicates the latest revision.
